# Assets Folder

This folder contains all image assets for the Vanish Trips website.

## Required Images

Place the following images in this folder:

1. **bali.jpg** - Image for Bali Escape trip (recommended: 800x600px or larger)
2. **tokyo.jpg** - Image for Tokyo Nights trip (recommended: 800x600px or larger)
3. **swiss-alps.jpg** - Image for Swiss Alps Adventure trip (recommended: 800x600px or larger)
4. **iceland.jpg** - Image for Iceland Northern Lights trip (recommended: 800x600px or larger)
5. **morocco.jpg** - Image for Morocco Discovery trip (recommended: 800x600px or larger)
6. **cambodia.jpg** - Image for Cambodia Temples trip (recommended: 800x600px or larger)

## Image Sources

You can use free stock photos from:
- Unsplash (https://unsplash.com)
- Pexels (https://www.pexels.com)
- Pixabay (https://pixabay.com)

Search terms:
- "Bali beach temple"
- "Tokyo night city"
- "Swiss Alps mountain"
- "Iceland aurora northern lights"
- "Morocco desert Marrakech"
- "Cambodia Angkor Wat temple"

## Image Format

- Format: JPG or PNG
- Recommended size: 1200x800px or larger
- Aspect ratio: 3:2 or 16:9
- File size: Optimize for web (under 500KB per image recommended)

## QR Code

If you have a QR code file (qr.png), place it in the root directory of the project (same level as index.html), NOT in the assets folder.
